# Blogger Box - Features Documentation

Below is a comprehensive list of the features and capabilities currently implemented in the application.

---
# You can :

- create artcile using the "write" button in the top right.
- create/delete/modify categories using the "write" button in the top right.
- edit/delete articles using the ✎ and ✕ buttons on each post card.
- read full articles by clicking their titles on the home page.
- search, filter (by category), and sort articles using the control bar on the home page.
